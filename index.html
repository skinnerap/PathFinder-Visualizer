How it Works

  • Select "Start" in the menu at the top of the page
  • Next, click any block on the screen to transform it into a "Start" block
  • Repeat the process for the "End" block
  • Optional: Select "Wall" and click as many blocks as you want to create barriers
  • If the algorithm uses it, select a heuristic function
  • Finally, click the algorithm button of your choice and enjoy!
  • Special Note: Clicking "Reset" after each execution resets all blocks completely!
